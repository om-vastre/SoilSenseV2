# SoilSenseV2

**New complete version of Soil Sense.**

## Overview
SoilSenseV2 is a Java-based application designed for soil monitoring and analysis. This repository contains the complete source code and implementation details for the updated system.

## Tech Stack
- **Language:** Java (100%)
- **BaaS:** Firebase (Auth, RealtimeDB, Firestore, Hosting, Storage)

## Getting Started

### Prerequisites
To run this project, ensure you have the following installed:
*   **Java Development Kit (JDK):** Version 8 or higher.
*   **IDE:** IntelliJ IDEA, Eclipse, or VS Code (recommended).

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/om-vastre/SoilSenseV2.git
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd SoilSenseV2
    ```

3.  **Compile and Run:**
    *   If using an IDE, import the project folder and run the main class.
    *   If using the command line:
        ```bash
        javac -d bin src/*.java
        java -cp bin MainClassName
        ```
        *(Note: Replace `MainClassName` with the actual name of your entry point class)*
